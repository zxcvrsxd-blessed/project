# ZooStore

Веб-сайт зоомагазина "Любимый хвостик" на Flask с каталогом, объявлениями и админкой.

## Локальный запуск

```bash
cd ~/PyCharmMiscProject/zoo_store
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python app.py
```

Админка:

```text
http://127.0.0.1:порт/admin
login: admin
password: admin123
```
Так же в самом низу правого угла сайта есть кнопка для входа в админку.

Пароль можно поменять через переменную `ADMIN_PASSWORD` до первого запуска проекта.


Так же я задеплоил проект на VPS-сервак через rsync и привязал к нему домен https://любимый-хвостик.рф, так что если нужна исключительно визуальная оценка без локального деплоя можно перейте по ссылке(логин пароль от админки тьам другой).
